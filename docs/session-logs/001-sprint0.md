- 曖昧指示「TODO タイトルのスラッグ関数」に対し、存在しない `str.slugify()` を

- **観察（実行）**: `AttributeError: 'str' object has no attribute 'slugify'`（exit=1）。
- **検出網**: 型チェック（最速・未整備）／実行（即停止）／テスト（CI で恒久）の3層を提示。
  追加依存なしで再現できる **実行・テスト** を実演し緑まで確認（`unittest` 3件 OK）。
- 赤（幻覚）→ 仕様追加 → 緑（検証網通過）の一往復を、以降の最小単位として定義。
